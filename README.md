# TwilmStream

Harvests tweets and user data related to movies from Twitter Stream API.

## Installation

Add this line to your application's Gemfile:

    gem 'twilm_stream'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install twilm_stream
